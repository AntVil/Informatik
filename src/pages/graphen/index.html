Graphen

Beschreibung

Ein Graph ist eine Relation auf einer Kantenmenge.
G = (M, R)
Ein gewichteter Graph besitzt für jede Kante der Kantenmenge ein Gewicht.
G = (M, R, g)
g:R \rightarrow \mathbb{R}

Depth-First-Search

Durchlaufen eines Graphen mit einem Stack, Knoten werden nur einmal besucht.

Breadth-First-Search

Durchlaufen eines Graphen mit einer Queue, Knoten werden nur einmal besucht.

Spannbaum

Algorithmus von Kruskal

Kanten Sortieren Kanten hintereinander hinzufügen Außer es entsteht ein Zyklus

Algorithmus von Prim

Knoten zu Spannbaum hinzufügen kleinste Kante angrenzend zum Baum hinzufügen Außer es entsteht ein Zyklus

Algorithmus von Dijkstra

Startknoten wird besucht Nachbarn und Distanzen von Knoten werden gegebenenfalls überschrieben Kleinster unbesuchter Nachbar wird besucht

Maximaler Fluss

Minimaler Schnitt

Es wird ein Schnitt durch nur vollständig verwendete Kanten gesetzt. Die Summe der geschnittenen Kanten entspricht dem Maximalem Fluss.
