Array} Ein Array ist eine geordnete Sammlung, wobei jedes Element schnell über einen Index abgerufen oder verändert werden kann. Arrays können mehrere Dimensionen haben, jedoch muss die Größe eines Arrays bei seiner Erstellung vorgegeben werden und kann nicht verändert werden.

Liste} Eine Liste ist eine flexiblere Datenstruktur, da die Größe dynamisch verringert oder erweitert werden kann. Listen basieren auf Knoten und Pointern, welche leichter verändert werden können. Die Pointer eines Listen Elements zeigen immer auf den Nachfolger in der Liste, so kann diese iterativ durchlaufen werden. \begin{center} \begin{tikzpicture} \fill[cyan!20, rounded corners] (0cm, 0cm) rectangle (1.5cm, 1cm); \fill[orange!20, rounded corners] (0cm, 0cm) rectangle (1cm, 1cm); \fill[cyan!20, rounded corners] (2.5cm, 0cm) rectangle (4cm, 1cm); \fill[orange!20, rounded corners] (2.5cm, 0cm) rectangle (3.5cm, 1cm); \fill[cyan!20, rounded corners] (5cm, 0cm) rectangle (6.5cm, 1cm); \fill[orange!20, rounded corners] (5cm, 0cm) rectangle (6cm, 1cm); \draw[cyan, ->] (1.25cm, 0.5cm) -- (2.5cm, 0.5cm); \draw[cyan, ->] (3.75cm, 0.5cm) -- (5cm, 0.5cm); \draw[cyan, ->] (6.25cm, 0.5cm) -- (7.5cm, 0.5cm); \end{tikzpicture} \end{center} Eine Liste kann auch bidirektional sein, die Pointer gehen in beide Richtungen. Dadurch kann die Listen einfach in beide Richtungen durchgegangen werden und wird noch flexibler. \begin{center} \begin{tikzpicture} \fill[cyan!20, rounded corners] (0cm, 0cm) rectangle (2cm, 1cm); \fill[orange!20, rounded corners] (0.5cm, 0cm) rectangle (1.5cm, 1cm); \fill[cyan!20, rounded corners] (3cm, 0cm) rectangle (5cm, 1cm); \fill[orange!20, rounded corners] (3.5cm, 0cm) rectangle (4.5cm, 1cm); \fill[cyan!20, rounded corners] (6cm, 0cm) rectangle (8cm, 1cm); \fill[orange!20, rounded corners] (6.5cm, 0cm) rectangle (7.5cm, 1cm); \draw[cyan, ->] (1.75cm, 0.75cm) -- (3cm, 0.75cm); \draw[cyan, ->] (4.75cm, 0.75cm) -- (6cm, 0.75cm); \draw[cyan, ->] (7.75cm, 0.75cm) -- (9cm, 0.75cm); \draw[cyan, ->] (0.25cm, 0.25cm) -- (-1cm, 0.25cm); \draw[cyan, ->] (3.25cm, 0.25cm) -- (2cm, 0.25cm); \draw[cyan, ->] (6.25cm, 0.25cm) -- (5cm, 0.25cm); \end{tikzpicture} \end{center}

Stack}
{0.66\textwidth} Ein Stack (auch Stapel oder Keller) ist eine sonder Form einer Liste, bei der nur am Anfang und Ende eingefügt und entfernt werden kann. Er funktioniert nach dem Last-in-First-out Prinzip (LIFO)

Queue}
{0.66\textwidth} Eine Queue (auch Schlange) ist eine sonder Form einer Liste, bei der nur am Ende eingefügt und am Anfang entfernt werden kann. Sie funktioniert nach dem First-in-First-out Prinzip (FIFO). Eine priority Queue funktioniert ähnlich wie eine normale Queue, jedoch haben Elemente eine Priorität, welche benutzt wird um beim Einfügen manche Elemente weiter vor zu ziehen, damit diese schneller bearbeitet werden.
